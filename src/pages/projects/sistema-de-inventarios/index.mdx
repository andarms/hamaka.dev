¡Te doy la bienvenida a este emocionante proyecto el que crearemos una aplicación de manejo de inventarios utilizando Node.js, MySQL y Angular! A lo largo de esta serie de lecciones y videos, te guiaré detalladamente en cada paso del proceso, desde el inicio hasta la puesta en producción de una aplicación completamente funcional.

He elegido crear un sistema de inventario porque es un problema común al que se enfrentan muchas empresas, y no se necesita ser un experto para entender el problema y aprender cómo solucionarlo. Durante el desarrollo del proyecto te mostraré cómo aplicar buenas prácticas de desarrollo para crear una aplicación robusta y sostenible. 

Si bien utilizaremos tecnologías específicas como Node.js con TypeScript para el back-end, MySQL como motor de base de datos y Angular para el front-end, lo que quiero que aprendas en este proyecto son las bases para resolver los desafíos y problemas que surjan al crear la aplicación. Te explicaré por qué se hacen las cosas de una determinada manera, por qué hemos decidido usar X o Y solución, y quiero que los conocimientos adquiridos sean transferibles a otros conjuntos de tecnologías.

Crearemos la aplicación de una manera ágil, entregando valor poco a poco en cada iteración. Pasaremos por el proceso de planificación, desarrollo, pruebas y puesta en producción, emulando a pequeña escala lo que podríamos encontrar en un entorno de producción real. Para cada hito (milestone), planificaremos los datos, desarrollaremos la funcionalidad y la pondremos en producción.


Aquí es donde quiero destacar un poco la diferencia con otros cursos, talleres y formaciones disponibles en la actualidad. Muchos de ellos solo se enfocan en escribir código, lo cual es solo una parte del ciclo de vida del software y descuidan varios aspectos que considero importantes de tener en cuenta, especialmente si eres alguien que está comenzando y aún no ha enfrentado los desafíos que conlleva llevar una aplicación a producción.

Este proyecto está dirigido a desarrolladores que buscan mejorar sus habilidades y aprender nuevas tecnologías. Aunque no es necesario ser un experto en Node.js, MySQL o Angular, es importante tener un conocimiento sólido de programación en JavaScript y comprender conceptos clave como variables, control de flujo, estructuras de datos, Programación Orientada a Objetos y otros aspectos similares. Es recomendable tener algo de familiaridad con TypeScript para seguir este proyecto de manera efectiva.

En resumen, este proyecto es una guía completa para aprender a desarrollar una aplicación de manejo de inventarios, donde abordaremos tanto las buenas prácticas de desarrollo como las diferentes etapas de producción. Ya teniendo en cuenta los conocimientos básicos de programación, estamos listos para comenzar con la primera fase del proyecto: **[Planeación inicial y definición del alcance](/projects/sistema-de-inventarios/001-planeacion-y-alcance).**. ¡Prepárate para empezar!
